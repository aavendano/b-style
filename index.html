Variables CSS

Clase Token dp equiv. Uso típico MD3
.elevation-0 --md-elevation-0 0 dp Superficies planas, fondos
.elevation-1 --md-elevation-1 1 dp Cards, text fields, list items
.elevation-2 --md-elevation-2 2 dp Contained buttons, cards hover
.elevation-3 --md-elevation-3 4 dp FAB, dropdowns, chip on drag
.elevation-4 --md-elevation-4 8 dp Navigation drawer, side sheets
.elevation-5 --md-elevation-5 12 dp Dialogs, navigation bar, modals

Uso con componentes Bulma

Card · elevation-1

Tarjeta en reposo. Sombra sutil de 1dp.

1 dp

Box · elevation-3

Dropdown, popover o panel flotante. 4dp de profundidad.

4 dp

Dialog · elevation-5

Sombra profunda de 12dp para diálogos modales.
